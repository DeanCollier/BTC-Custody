Bitcoin Custody

Now that we have covered what bitcoin is, lets go over the different ways an individual can custody their bitcoin! In order to understand the different custody options, we need to first understand what a bitcoin address and what a bitcoin wallet are.


Address

  1. Private Key - a private key represents a randomly generated 256 bit number that is unique (see below for all posibilities)
  2. Public Key - a public key is an encrypted version of your private key that you can share with others to recieve bitcoin

Wallet

  1. Key Storage - a wallet stores your private key, the private key is needed to sign transactions that you make
  2. Blockchain Interaction - a wallet uses your bitcoin address to display how much bitcoin you own
  3. Transactions - wallets allow you to send and recieve bitcoin
